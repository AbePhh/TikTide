# TikTide

TikTide 是一个以 Golang 为后端、网页端为前端的短视频平台练手项目。

当前仓库聚焦以下内容：

- `backend/`：后端项目目录骨架，按网关、用户、视频、互动、关注流等服务划分
- `frontend/`：单网页端项目目录骨架
- `文档/`：项目设计、数据库、接口和开发规范文档

## 顶层目录说明

### `backend/`
后端主目录，保存后端服务骨架、公共包、脚本和测试目录。

### `frontend/`
前端主目录，当前为单网页端项目结构，不再区分移动端和多应用 monorepo。

### `文档/`
项目设计文档目录，保存架构、数据库接口、开发规范和配图文件。
